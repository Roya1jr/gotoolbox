# GoToolbox

A collection of useful Go utilities and tools for common programming tasks.

## Installation

```bash
go get github.com/Roya1jr/gotoolbox
```

## Quick Start

```go
package main

import (
 "log"
 "github.com/Roya1jr/gotoolbox/gtb_string"
)

func main() {
 log.Println(gtb_string.AddPre("World", "Hello"))
}
```
